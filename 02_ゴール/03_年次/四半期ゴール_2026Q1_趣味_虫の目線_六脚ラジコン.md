# 四半期ゴール_2026Q1_趣味_虫の目線_六脚ラジコン

## 1. この四半期のテーマ
- “電源が落ちない12DOF六脚” をまず成立させる（映像は後追いでもよいが、拡張余地は残す）。

## 2. 成果指標（KGI）
- KGI-1: AA電池（アルカリ or NiMH）で、Pi + XIAO + サーボ駆動を同時に行っても致命的なリセットが起きない。
- KGI-2: 六脚が、前進/後退/旋回/停止をできる（転倒しても原因が説明できるログ/再現手順がある）。
- KGI-3: 通信断・異常時に安全停止する（フェイルセーフ確認）。

## 3. 行動指標（KPI）
- 毎週「前進デモ」まで触る（机上設計だけにしない）。
- 設計判断メモ（電源レール、セル数、保護、配線）を更新する。

## 4. 今期のスコープ（やる/やらない）
- やる
  - 電源設計（AA電池前提）と実機での安定化
  - 12サーボの駆動と基本歩容（最小限）
- やらない（後回し）
  - 低遅延の追い込み（まずは“見える”でよい）
  - 外装の作り込み（安全最低限はやる）
