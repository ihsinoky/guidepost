
## 1. 年ごとのゴール一覧

```dataview
TABLE
  area as "分野",
  level as "レベル",
  link(file.name) as "目標",
  status as "状態"
FROM "02_ゴール"
WHERE type = "goal" AND year >= 2025 AND year <= 2030
GROUP BY year
SORT year asc, area asc
```


## 2. 分野ごとの年次ゴール

```dataview
TABLE
  year as "年",
  level as "レベル",
  link(file.name) as "目標",
  status as "状態"
FROM "02_ゴール/03_年次"
WHERE type = "goal" AND level = "年次"
GROUP BY area
SORT area asc, year asc
```



## 3. 投資一覧

```dataview
TABLE
  service as "サービス",
  category as "種別",
  area_main as "主な分野",
  cost_month as "月額",
  status as "状態"
FROM "04_投資とサブスク"
WHERE type = "investment"
SORT area_main asc, service asc
```
