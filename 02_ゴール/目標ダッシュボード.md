
## 1. 年ごとのゴール一覧

```dataview
TABLE
  year as "年",
  quarter as "四半期",
  area as "分野",
  level as "レベル",
  file.link as "目標",
  status as "状態"
FROM "02_ゴール"
WHERE type = "goal"
SORT year asc, quarter asc, area asc, file.name asc

```


## 2. 投資一覧

```dataview
TABLE
  service as "サービス",
  category as "種別",
  area_main as "主な分野",
  cost_month as "月額",
  status as "状態"
FROM "04_投資とサブスク"
WHERE type = "investment"
SORT area_main asc, service asc
```
